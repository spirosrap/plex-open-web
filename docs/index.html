Plex Open Web

A small self-hosted Plex web UI for private browser playback, library browsing, subtitle selection, OpenSubtitles downloads, and tailnet access without relying on Plex cloud remote playback.

Plex Open Web

Private remote playback

Serve your own authenticated web app through Tailscale or another private network.

Subtitle workflow

Use Plex subtitle streams, sidecar files, and optional OpenSubtitles search/download.

Browser compatibility

Use FFmpeg to convert unsupported audio to AAC when the browser cannot play the source.
